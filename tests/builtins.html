Builtins

Builtins are core builtin functions of Rye language

join

Pure Builtin(1): Joins Block or list of values together.

join { "abc" "dfe" "ghi" }
; returns "abcdfeghi"
{ "abc" 123 } .join
; returns "abc123"

length?

Builtin(1): Accepts a collection (String, Block, Dict, Spreadsheet) and returns it's length.

length? "1234567"
; returns 7
{ 1 2 3 4 5 } .length?
; returns 5
list { 1 2 3 } |length?
; returns 3

map

Pure Builtin(2): Maps values of a block to a new block by evaluating a block of code.

map { 1 2 3 } { + 1 }
; returns 2 3 4
map { "a" "bb" "ccc" } { .length? }
; returns 1 2 3
